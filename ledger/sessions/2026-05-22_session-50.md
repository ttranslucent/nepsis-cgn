## Boundary Decisions Captured

- The new stateless MCP operator packet runtime is canonical for user-owned
  model connectivity.
- Existing stateful `/v1/operator/*` routes remain private and unchanged for
  the current operator UI and compatibility. They are transitional for the
  user-owned model connectivity path.
- NepsisCGN does not collect OpenAI, Anthropic, Gemini, or other provider keys
  for the MCP path.
- Hosted model routes remain disabled by default and separate from MCP.
- Capability tokens are Nepsis access tokens only; store hashes and metadata,
  never provider keys.
